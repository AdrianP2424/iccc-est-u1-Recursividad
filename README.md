## PC 1.3 Ejercicios 1 a 3
### DESCRIPCION:

Se creó una clase en Java llamada Ejercicios que implementa varios métodos recursivos. En ella se desarrollan funciones para sumar números consecutivos desde un valor dado hasta 1, mostrando en uno de los casos el proceso paso a paso, y también se incluye un método para calcular la potencia de un número. Todos los métodos utilizan recursividad, definiendo un caso base y llamándose a sí mismos para resolver el problema progresivamente.
